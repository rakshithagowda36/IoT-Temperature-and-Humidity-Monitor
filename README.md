# Circuit Diagram
The circuit diagram of the IoT Temperature and Humidity Monitor includes:
- ESP8266 NodeMCU
- DHT11 Sensor
- Breadboard
- Jumper Wires
- Power Supply
The sensor is connected to the ESP8266 to measure temperature and humidity data and send it to the cloud platform.
# Debugging Process
During project development, several debugging steps were performed:
- Checked sensor connections properly
- Verified WiFi connectivity
- Corrected errors in Arduino IDE code
- Monitored serial output for data validation
- Tested sensor readings multiple times
These debugging steps helped improve the accuracy and reliability of the project.
# Graph Visualization
The collected temperature and humidity data were visualized using graphs in the IoT dashboard.
The graph visualization helps to:
- Monitor real-time sensor data
- Analyze temperature changes
- Track humidity variations over time
- Improve environmental monitoring
